The Excel Data Masker is a modular Python app that anonymizes sensitive Excel data through a user-friendly interface.

Modules:

  • main_app.py – Launch UI and main loop
  • ui_field_selector_grid.py – Grid-based field selector (tksheet)
  • mask_logic.py – Field-by-field masking logic

📁 Folder Structure

  • /docs – GitHub Pages HTML and architecture documentation
  • /sample_data – Example input spreadsheets
  • /.github/workflows – GitHub Actions CI for linting
  • main_app.py, requirements.txt, README.md, etc. – Core project files

🔐 Privacy and Security

  • No tracking, telemetry, or remote access features
